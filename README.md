# n8n_ai_agents
AI workflow automation projects developed during the Bharat Unnati AI Fellowship using n8n, Generative AI, LLMs, and Prompt Engineering. Includes AI-powered Ad Copy Scoring and Lead Qualification Agents with webhook integration, cloud automation, and intelligent business workflow solutions.

## Features

* AI-powered workflow automation
* Ad Copy Scoring Agent
* Lead Qualification Agent
* Webhook and cloud integrations
* Google Sheets automation
* Prompt Engineering techniques
* Intelligent business process automation

## Technologies Used

* n8n
* Generative AI
* Large Language Models (LLMs)
* Prompt Engineering
* Google Sheets
* Webhooks
* AI Chains

## Learning Outcomes

* Understanding AI and workflow automation
* Building AI-powered agents
* Practical exposure to cloud integrations
* Improving analytical and problem-solving skills
* Learning real-world AI applications
